# **App Name**: medicoAI

## Core Features:

- Responsive Layout: Responsive layout that works perfectly on mobile, tablet, and desktop devices.
- Chat Interface: Chat interface with clear distinction between user and bot messages, styled with custom Tailwind CSS classes.
- Typing Indicator: Typing effect and indicator to simulate a real-time conversation with the bot.
- Markdown Support: Markdown rendering in bot responses to support tables, lists, code snippets, math formulas, and more.
- Copy Button: Small 'copy' button next to bot responses for easy text copying.
- Image Upload and Analysis: Image upload alongside text input for vision analysis. Display images inline within chat history before sending to the Pollination AI API, leveraging its 'vision' tool for analysis and explanation using models such as 'openai' and 'mistral'.
- Settings Customization: Settings modal/drawer to change theme colors, font size/style, all live-updating via Tailwind CSS classes.
- Animated Splash Screen: Animated splash screen/logo at startup.

## Style Guidelines:

- Primary color: HSL(142, 57%, 48%) - Forest green (#49c140). Evokes a sense of health and growth.
- Background color: HSL(150, 17%, 96%) - Very light green (#f4f8f5). Provides a soft, calming backdrop.
- Accent color: HSL(112, 51%, 53%) - Lime green (#87e550). To draw attention and complement the green primary.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable interface.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets with clarity.
- Integrate relevant icons (Font Awesome or similar) for chat bubbles, users, settings, upload, etc. to enhance user experience.
- Utilize rounded corners and smooth shadows/effects for depth, creating a modern and friendly user interface.
- Incorporate subtle animations and transitions to enhance user engagement without overwhelming the interface.
