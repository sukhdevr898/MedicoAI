📚 Endpoint Documentation

Base URL: Your server URL (e.g., http://localhost/mcp.php or https://yourserver.com/mcp.php)

  • Health Check: GET /mcp.php/health
  • URL Parameters: GET /mcp.php?prompt=question&model=openai
  • Route-based: GET /mcp.php/ask/your-question
  • Model Route: GET /mcp.php/model/gemini/your-question
  • JSON POST: POST /mcp.php with JSON body
  • Debug Mode: Add ?debug=1 to any request
